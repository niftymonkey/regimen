# Regimen is a discipline embodied in pluggable instruments

> Status: accepted (2026-05-18)

Regimen's object is the interaction discipline between an engineer and an AI agent. That discipline is embodied in pluggable instruments, adopted individually as felt needs arise, rather than delivered as a document to read and adopt wholesale, because engineers will not stop work to internalize a practice before using it. Two viewpoints hold at once: the adopting engineer experiences only pluggable tools, while the design must hold the discipline as the spine that gives those tools coherence.

## Considered options

- Delivering the practice as a document the engineer reads and follows. Rejected: engineers do not stop work to internalize and execute a document with fidelity.
- A pure software system of independent tools with no connecting discipline. Rejected: instruments designed with no shared model of good practice surface noise rather than feedback.
